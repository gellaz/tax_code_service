Welcome to the Tax Code Service!

JSON API that allows a client to extrapolate person's data from a tax code string and viceversa.

Get person's data from tax code

Send a GET request to localhost:5000/api/<TAX_CODE>

Example: localhost:5000/api/RSSMRA75L01H501A

Get tax code from person's data

Send a POST request to localhost:5000/api/ with a request body including person's data in this format: { "surname": "string", "name": "string", "sex": "string", "birthdate": "string", "birthplace": "string" }

Example:

Enter birth date in the form dd/mm/YYYY or YYYY-mm-dd
